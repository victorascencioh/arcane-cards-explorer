# Arcane Cards Explorer

Arcane Cards Explorer is a modern web application for exploring Magic: The Gathering cards using the Scryfall API.

## Features (Planned)

* Card search
* Favorites
* Deck builder
* Deck simulator
* User authentication
* Deck sharing

## Tech Stack

* Angular
* Signals
* NgRx
* Firebase
* TailwindCSS
* Jest
* Cypress

## API

Scryfall API
https://scryfall.com/docs/api
